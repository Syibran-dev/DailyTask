DailyTask adalah aplikasi mobile berbasis Android yang digunakan untuk membantu pengguna dalam mengelola dan mencatat tugas harian secara sederhana dan efektif. Aplikasi ini memungkinkan pengguna untuk menambahkan, melihat, mengedit, dan menghapus daftar tugas, sehingga aktivitas harian dapat terorganisir dengan lebih baik.
Aplikasi DailyTask dikembangkan menggunakan bahasa pemrograman Kotlin dan SQLite sebagai media penyimpanan data lokal. Dengan antarmuka yang sederhana dan mudah digunakan, DailyTask dirancang agar pengguna dapat dengan cepat mengelola tugas tanpa proses yang rumit.
Fitur Utama:
1.Menambahkan tugas baru dengan judul, deskripsi, dan deadline
2.Menampilkan daftar tugas menggunakan RecyclerView
3.Menandai tugas sebagai selesai
4.Mengedit dan menghapus tugas
5.Penyimpanan data secara lokal menggunakan SQLite

Teknologi yang Digunakan:
1.Android Studio
2.Kotlin
3.SQLite Database
4.RecyclerView
Aplikasi ini dibuat sebagai bagian dari tugas Pemrograman Berbasis Mobile dan dikembangkan secara berkelompok (maksimal 2 orang).
